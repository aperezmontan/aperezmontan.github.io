What is pair programming?

Pair programming is a collaboration between 2 or more coders, whether we're right next to each other or working remotely through stypi and/or Google hangouts, etc.

Why is pair programming important to DBC?

Pair programming is a necessary part of DBC and helps speed up development. We get to debug each other's code.
